# Get-UnifiAdminUsers

## SYNOPSIS
Retrieves Unifi Controller admin users.

## SYNTAX

```
Get-UnifiAdminUsers
```

## DESCRIPTION
The Get-UnifiAdminUsers function retrieves administrative users from the Unifi Controller.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-UnifiAdminUsers
```

This example retrieves the administrative users.
